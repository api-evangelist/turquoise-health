# Turquoise Health

Healthcare price transparency platform with a REST API for accessing hospital
and insurer negotiated rates, out-of-pocket cost estimates, and
machine-readable files data.

**Website:** https://turquoise.health/  
**API Docs:** https://turquoise.health/api/redoc  
**GitHub:** https://github.com/turquoisehealth  
**Blog:** https://turquoise.health/resources/blog/  
**LinkedIn:** https://www.linkedin.com/company/turquoise-health/  
**X / Twitter:** https://twitter.com/TurquoiseHC  

## APIs

- **Turquoise Health REST API** — Core API for negotiated rates, MRF data, and cost estimates
- **Turquoise Connector (MCP)** — MCP server for AI tool integration with healthcare pricing data
- **Standard Service Packages (SSP) API** — Free API for standardized healthcare procedure codes
